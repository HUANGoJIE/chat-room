# chat-room
一款基于C/S架构，Java语言编写的网络聊天室。

### 问题陈述
实现一个聊天程序

### 功能分析
- 用户注册
- 用户登陆
- 一个聊天室
- 多个客户同时聊天
- 不能重复登陆
- 每个人能够看到当前聊天室的情况，有多少人，都有哪些人在聊天。
- 多个聊天室
- 能够私聊

### 技术点
- 语言基础知识
- 网络通信
- GUI
- 多线程
- JDBC数据库访问
- I/O
- 一些基本类库（时间，字符串处理等等）
